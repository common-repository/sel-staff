== Description ==
This plugin registers a custom post type for staff items, taxonomy, widgets and metabox (cmb2).

*This plugin output the plain content without style.*
*This plugin required 'CMB2' plugin*

== Frequently Asked Questions ==

= Installation Details =
1. Upload 'sel-staff' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Install required plugin, called 'CMB2'.
1. Add your staffs.

=Can I use this plugin with “non-selthemes's” themes? =
No, this plugin doesn't show content itself.
